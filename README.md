## 🚀 Overview

Welcome to the **Slooze Frontend Project**. This application is built with peak performance and cinematic design in mind, featuring:

- **Next.js 15+** - For powerful, optimized React rendering.
- **Framer Motion** - Delivering buttery-smooth, carefully choreographed animations.
- **Tailwind CSS** - Providing a gorgeous, utility-first styling system.

### The Signature Login Animation

This repository houses the meticulously crafted signature login experience:
- A seamless, non-reloading transition into the dashboard.
- Elegant, synchronized panel sliding that pushes the boundary.
- A dramatic text scale and cinematic screen-blur, blending perfectly into the target page.

---

## 💻 Getting Started

1. Clone this repository locally.
2. Install the necessary packages:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) with your browser to experience the portal.
